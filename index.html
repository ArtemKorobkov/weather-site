Возможности

Всё, что нужно для погоды

Полный набор функций для мониторинга и анализа погодных условий

🌡️

Текущая погода

Температура, ощущаемая температура, влажность, скорость ветра, давление, видимость, восход и закат — данные из OpenWeatherMap API в реальном времени.

📅

Прогноз на 5 дней

Детальный прогноз с шагом 3 часа (до 40 записей). Текстовый график изменения температур. Сохранение прогнозов в текстовые файлы.

🧮

Погодный калькулятор

Конвертация температур: Цельсий → Фаренгейт → Кельвин. Расчёт комфортности с учётом температуры, ветра и влажности. Индекс ветро-холода и индекс жары.

📍

Геолокация по IP

Автоматическое определение вашего местоположения по IP-адресу. Мгновенная подстановка города для получения погоды без ручного ввода.

💾

Кэширование и история

SQLite база данных: кэширование погоды на 10 минут, прогноза на 30 минут. История всех запросов с временем. Настройки сохраняются между запусками.

🎨

Тёмная и светлая тема

Современный градиентный интерфейс с glassmorphism-эффектами. Переключение тем с сохранением настроек в базу данных. Плавные анимации.

Десктопное приложение

Мощный GUI на Flet

Кроссплатформенное приложение с современным интерфейсом, построенное на Python и фреймворке Flet. Работает на Windows, macOS и Linux.

  • Живые часы и дата с автообновлением
  • Интерактивные карточки погоды
  • Текстовый график температур
  • История запросов с очисткой
  • Сохранение прогнозов в файлы
  • Случайные советы о погоде
  • Обработка всех ошибок без вылетов
Telegram-бот

Погода в кармане 24/7

Полноценный Telegram-бот на aiogram 3.x, развёрнутый на Railway. Доступен всем пользователям — узнавайте погоду без установки приложений.

  • Команды: /weather, /forecast, /convert, /comfort
  • Сравнение погоды двух городов
  • Погода по геолокации (GPS)
  • Случайные факты о погоде (/fact)
  • Погодный гороскоп (/horoscope)
  • Rate limiting для защиты от спама
  • Inline-кнопки для быстрого доступа
